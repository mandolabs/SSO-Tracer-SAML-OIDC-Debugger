# Security Policy

## Supported Versions
Only the latest release is supported with security updates.

## Reporting a Vulnerability
Please report security issues privately by opening a GitHub Security Advisory:
https://github.com/mandolabs/SSO-Tracer-SAML-OIDC-Debugger/security/advisories

If advisories are not available, open a regular issue and clearly mark it as
SECURITY so we can move the discussion to a private channel.

## Scope
This project is a browser extension intended for SSO troubleshooting.
Please do not include real credentials, tokens, or unredacted SAML responses
in reports.
